# Tenhou Score Manager (天鳳成績管理)

Tenhou Score Manager は、オンライン麻雀「天鳳」の対局成績を記録・分析するためのローカル完結型Webアプリケーションです。
ブラウザのローカルストレージ（IndexedDB）を活用し、高速かつプライバシーを保護した状態でデータを管理します。洗練されたダッシュボードと自動インサイト（分析）機能により、日々の成績向上をサポートします。

## 主な機能

- **完全ローカル保存**
  すべての対局データはブラウザ内（IndexedDB）に保存されます。外部サーバーとの通信は一切なく、動作は軽量です。（JSON形式でのバックアップ機能も搭載しています）

- **インサイトカード機能（自動成績分析）**
  直近の成績データを分析し、好調・不調の波や課題を自動で言語化してカード形式で表示します。（例：「🔥 3連続トップ」「⚠ 直近50戦 ラス率高め」など）

- **多角的な集計・分析**
  - **基本集計**: 全体・東風・東南のルール別にトップ率、連対率、平均順位、安定段位などの詳細指標を算出
  - **推移グラフ**: レーティング（R）やポイントの変動履歴をチャートで可視化
  - **期間別集計**: 日別・週別・月別での成績推移を一覧表示
  - **対戦数別集計**: 10戦/50戦/100戦ごとなど、任意の対局数区切りで集計。最新対局からの遡りにも対応

- **モダンなUI & ダークモード対応**
  直感的な操作が可能なダッシュボードデザインを採用。ワンクリックで目に優しいダークモードへの切り替えも可能です。

##  技術スタック

- **Framework**: React 18 / TypeScript / Vite 8
- **Styling**: Tailwind CSS v4
- **Database**: Dexie.js (IndexedDB)
- **Charts**: Recharts
- **Icons**: Lucide React

##  ローカル環境での起動方法

Node.js (`v18` 以上推奨) がインストールされている環境で、以下のコマンドを実行してください。

```bash
# 1. 依存パッケージのインストール
npm install

# 2. 開発サーバーの起動
npm run dev

# 3. 本番用ビルド（必要に応じて）
npm run build
```

起動後、コンソールに表示される URL（例: `http://localhost:5173`）をブラウザで開いてアクセスしてください。

##  使い方

1. **初期設定**
   「初期設定」タブを開き、あなたの現在の「段位」「ポイント」「レーティング」を登録します。（ここがグラフなどの起点になります）
2. **対局の記録**
   画面上部の入力フォームから、対局ルール（東風/東南）、順位、ポイント変動、R変動を入力して追加します。
3. **分析の確認**
   ダッシュボードから、インサイトカードや各集計データを確認し、成績の振り返りに活用してください。

##  注意事項

- 本アプリのデータはブラウザのローカル領域に保存されます。**ブラウザの「閲覧データの消去」や「サイトデータの削除」を行うと、記録したデータも完全に消失します**のでご注意ください。
- 大切なデータは、初期設定画面の「エクスポート」機能を利用して、定期的にバックアップ（JSONファイルの保存）を行ってください。
